# ตัวอย่าง Chat — Wikipedia Tool

## คำถามที่ AI ควรเรียก Wikipedia

### ค้นหาข้อมูลที่ LLM อาจไม่รู้
```
User: Moo Deng คืออะไร
AI: [เรียก Wikipedia] หมูเด้งเป็นฮิปโปโปเตมัสแคระ (pygmy hippopotamus)
    ที่สวนสัตว์เปิดเขาเขียว จังหวัดชลบุรี ประเทศไทย ...
```

### ค้นหาข้อมูลประเทศ
```
User: ประเทศญี่ปุ่นมีประชากรเท่าไหร่
AI: [เรียก Wikipedia] ประเทศญี่ปุ่นมีประชากรประมาณ 123.4 ล้านคน ...
```

### ค้นหาข้อมูลวิทยาศาสตร์
```
User: ดาวอังคารอยู่ห่างจากโลกเท่าไหร่
AI: [เรียก Wikipedia] ดาวอังคารอยู่ห่างจากโลกโดยเฉลี่ยประมาณ 225 ล้านกิโลเมตร ...
```

## หมายเหตุ

- Wikipedia tool ค้นหาเป็น **ภาษาอังกฤษ**
- AI จะแปลคำค้นเป็นภาษาอังกฤษให้เอง
- ผลลัพธ์ AI จะแปลกลับเป็นไทยถ้ามี System Prompt บอกให้ตอบภาษาไทย
