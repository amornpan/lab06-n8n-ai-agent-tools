# ตัวอย่าง Chat — Multi-tool (Wikipedia + Calculator)

## คำถามที่ AI ต้องใช้ทั้ง 2 tools

### ตัวอย่างที่ 1: ประชากรลดลง
```
User: ประชากรญี่ปุ่นมีกี่คน ถ้าลดลง 2% ต่อปี อีก 5 ปีจะเหลือเท่าไหร่

AI ทำงาน:
  Step 1: [เรียก Wikipedia] ค้นหา "Japan population" → ได้ 123.4 ล้านคน
  Step 2: [เรียก Calculator] 123400000 * (0.98)^5 → ≈ 111.5 ล้านคน

AI: ปัจจุบันประชากรญี่ปุ่นอยู่ประมาณ 123.4 ล้านคน
    ถ้าลดลง 2% ต่อปี อีก 5 ปีจะเหลือประมาณ 111.5 ล้านคน
```

### ตัวอย่างที่ 2: GDP เติบโต
```
User: GDP ของไทยเท่าไหร่ ถ้าโต 3% ต่อปี อีก 10 ปีจะเป็นเท่าไหร่

AI ทำงาน:
  Step 1: [เรียก Wikipedia] ค้นหา "Thailand GDP" → ได้ ~$500 billion
  Step 2: [เรียก Calculator] 500 * (1.03)^10 → ≈ $672 billion

AI: GDP ของไทยปัจจุบันประมาณ 500 พันล้านดอลลาร์
    ถ้าเติบโต 3% ต่อปี อีก 10 ปีจะเป็นประมาณ 672 พันล้านดอลลาร์
```

### ตัวอย่างที่ 3: ทดสอบ Memory
```
User: ฉันชื่ออาร์ม
AI: สวัสดีครับคุณอาร์ม! มีอะไรให้ช่วยไหมครับ?

User: ฉันชื่ออะไร
AI: คุณชื่ออาร์มครับ (AI จำได้จาก Simple Memory)
```
