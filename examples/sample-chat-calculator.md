# ตัวอย่าง Chat — Calculator Tool

## คำถามที่ AI ควรเรียก Calculator

### คำนวณพื้นฐาน
```
User: 1234567 * 7654321 เท่ากับเท่าไหร่
AI: [เรียก Calculator] ผลลัพธ์คือ 9,449,772,114,007
```

### คำนวณเปอร์เซ็นต์
```
User: ถ้าเงินเดือน 35000 จ่ายภาษี 5% เหลือเท่าไหร่
AI: [เรียก Calculator] เงินเดือนหลังหักภาษี 5% = 35000 - (35000 * 0.05) = 33,250 บาท
```

### ยกกำลัง
```
User: 2 ยกกำลัง 32 เท่ากับเท่าไหร่
AI: [เรียก Calculator] 2^32 = 4,294,967,296
```

## วิธีดูว่า tool ถูกเรียก

1. ดูที่ **Logs** ใน n8n
2. Calculator node จะมี **checkmark สีเขียว** ✅
3. แสดง **items total** ที่ผ่าน Calculator
